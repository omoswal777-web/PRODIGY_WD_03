# 🎮 Tic Tac Toe Web Application

This is a Tic Tac Toe game built using HTML, CSS, and JavaScript.

## 🚀 Features
- Two player game (X vs O)
- Win detection
- Draw detection
- Score tracking
- Reset game & reset score

## 🛠 Tech Used
- HTML
- CSS
- JavaScript
